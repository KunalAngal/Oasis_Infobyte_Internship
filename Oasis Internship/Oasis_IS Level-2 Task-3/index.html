Instructions about working of Website


1. Write the task to be done in the Add Tasks section and click on "+" to add the new tasks.


2. If you are preponing or doing any particular task then you can just strikeout the task by single click on text.


3. If you have completed the task and you have to delete it then simply double clicking on the text so that it will be deleted and you will get the remaining task list in which the tasks are pending.
